# Archived Files

This folder contains files that are not needed for GitHub Pages hosting but are kept for reference or potential future use.

## Contents

- **bootstrap/**: Unused Bootstrap files (non-minified, source maps, theme files, config)
- **font-awesome/**: Unused Font Awesome files (SCSS, LESS, metadata, SVGs, sprites, JS)
- **demo-images/**: Theme preview images from original template
- **dev-files/**: Local development files (docker-compose.yml, minified JS)

## Note

These files are excluded from Jekyll build via `_config.yml`.
To restore any file, move it back to its original location.
