Shell Description

fish User-friendly shell

5. Using Help¶

man (Manual Pages)¶

# View command manual
man ls
man cp
man chmod

# Search by keyword
man -k "copy file"

Navigating man pages: | Key | Action | |-----|--------| | Space / f | Next page | | b | Previous page | | /search_term | Search | | n | Next search result | | q | Quit |

--help Option¶

# Quick help
ls --help
cp --help

info¶

# Detailed information (GNU commands)
info ls

6. Basic Commands¶

whoami - Current User¶

whoami

Output:

ubuntu

hostname - System Name¶

hostname

Output:

my-server

date - Current Date/Time¶

# Current date/time
date

# Specify format
date "+%Y-%m-%d %H:%M:%S"

Output:

Tue Jan 23 14:30:00 KST 2024
2024-01-23 14:30:00

cal - Calendar¶

# Current month
cal

# Specific year
cal 2024

# Specific month
cal 3 2024

clear - Clear Screen¶

clear
# Or Ctrl + L

echo - Print Text¶

echo "Hello, Linux!"
echo $HOME
echo "Current path: $(pwd)"

7. Command History¶

history Command¶

# View history
history

# Last 10 commands only
history 10

# Search history
history | grep "apt"

Using History¶

Command Description
!! Execute previous command
!n Execute nth history command
!string Execute most recent command starting with string
Ctrl + R Reverse history search 
# Re-execute previous command
!!

# Re-execute with sudo
sudo !!

# Execute command 123
!123

# Most recent command starting with ls
!ls

Ctrl + R (Reverse Search)¶

# Press Ctrl + R then enter search term
(reverse-i-search)`apt': apt update

# Enter: execute
# Ctrl + R: next result
# Ctrl + G: cancel
