단락과 텍스트 서식

이것은 단락(paragraph)입니다. HTML에서 텍스트를 그룹화합니다.

텍스트 서식 태그들:

  • strong - 중요한 텍스트 (굵게)
  • em - 강조 텍스트 (기울임)
  • mark - 하이라이트
  • del - 삭제된 텍스트
  • ins - 추가된 텍스트
  • code - 코드
  • H2O - 아래 첨자
  • x2 - 위 첨자

목록

순서 없는 목록 (ul)

  • 사과
  • 바나나
  • 오렌지

순서 있는 목록 (ol)

  1. 첫 번째 단계

HTML

인라인 인용 (q)

Steve Jobs는 Stay hungry, stay foolish라고 말했습니다.

div와 span

div는 블록 레벨 컨테이너입니다.

여러 요소를 그룹화할 때 사용합니다.

span은 인라인 컨테이너입니다. 텍스트 일부에 스타일을 적용할 때 사용합니다.

HTML 엔티티 (특수 문자)

  • < - 보다 작음 (&lt;)
  • > - 보다 큼 (&gt;)
  • & - 앰퍼샌드 (&amp;)
  •   - 공백 (&nbsp;)
  • © - 저작권 (&copy;)
  • ® - 등록상표 (&reg;)
  • € - 유로 (&euro;)

HTML 주석

주석은 브라우저에 표시되지 않습니다.

소스 코드를 확인하면 주석을 볼 수 있습니다.

코드 표시

function hello() {
    console.log("Hello, World!");
}
