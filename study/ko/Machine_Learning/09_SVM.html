2.2 ì†Œí”„íŠ¸ ë§ˆì§„ SVM¶

"""
ì†Œí”„íŠ¸ ë§ˆì§„ (Soft Margin):
- ì¼ë¶€ ì˜¤ë¶„ë¥˜ í—ˆìš©
- ìŠ¬ë™ ë³€ìˆ˜(Î¾) ë„ì…
- ì‹¤ì œ ë°ì´í„°ì— ì ìš© ê°€ëŠ¥

ìµœì í™” ë¬¸ì œ:

- C ì‘ìŒ: ì˜¤ë¶„ë¥˜ í—ˆìš© â†’ ë„“ì€ ë§ˆì§„, ì¼ë°˜í™” í–¥ìƒ
"""

# C íŒŒë¼ë¯¸í„° íš¨ê³¼

from sklearn.model_selection import train_test_split

# ë…¸ì´ì¦ˆê°€ ìˆëŠ” ë°ì´í„°
X, y = make_classification(
    n_samples=200, n_features=2, n_redundant=0,
    n_informative=2, n_clusters_per_class=1,
    flip_y=0.1,  # 10% ë…¸ì´ì¦ˆ
    random_state=42
)

    clf.fit(X, y)

    # ê²°ì • ê²½ê³„

    ylim = [X[:, 1].min() - 0.5, X[:, 1].max() + 0.5]

                         np.linspace(ylim[0], ylim[1], 100))

    Z = clf.decision_function(np.c_[xx.ravel(), yy.ravel()])

    ax.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')

               linestyles=['--', '-', '--'])
    ax.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='black')
    ax.scatter(clf.support_vectors_[:, 0], clf.support_vectors_[:, 1],
               s=150, facecolors='none', edgecolors='green', linewidths=2)
    ax.set_title(f'C = {C}\nSupport Vectors: {len(clf.support_vectors_)}')

plt.tight_layout()
plt.show()

3. ì»¤ë„ íŠ¸ë¦­ (Kernel Trick)¶

3.1 ì»¤ë„ í•¨ìˆ˜ì˜ ì›ë¦¬¶

"""
ì»¤ë„ íŠ¸ë¦­:
- ë¹„ì„ í˜• ë°ì´í„°ë¥¼ ê³ ì°¨ì› ê³µê°„ìœ¼ë¡œ ë§¤í•‘í•˜ì—¬ ì„ í˜• ë¶„ë¦¬
- ì‹¤ì œë¡œ ê³ ì°¨ì› ë³€í™˜ ì—†ì´ ë‚´ì ë§Œìœ¼ë¡œ ê³„ì‚° (íš¨ìœ¨ì )
- K(x, y) = Ï†(x)Â·Ï†(y)

ì£¼ìš” ì»¤ë„ í•¨ìˆ˜:

1. ì„ í˜• ì»¤ë„ (Linear):
   K(x, y) = xÂ·y
   - ì„ í˜• ë¶„ë¦¬ ê°€ëŠ¥í•œ ë°ì´í„°

2. ë‹¤í•­ì‹ ì»¤ë„ (Polynomial):
   K(x, y) = (Î³ * xÂ·y + r)^d
   - d: ë‹¤í•­ì‹ ì°¨ìˆ˜

3. RBF ì»¤ë„ (Radial Basis Function, Gaussian):
   K(x, y) = exp(-Î³||x - y||Â²)
   - ê°€ì¥ ë„ë¦¬ ì‚¬ìš©
   - ë¬´í•œ ì°¨ì›ìœ¼ë¡œ ë§¤í•‘

4. ì‹œê·¸ëª¨ì´ë“œ ì»¤ë„:
   K(x, y) = tanh(Î³ * xÂ·y + r)
   - ì‹ ê²½ë§ê³¼ ìœ ì‚¬
"""

3.2 ë¹„ì„ í˜• ë°ì´í„°ì— ì»¤ë„ ì ìš©¶

from sklearn.datasets import make_moons, make_circles

# ë¹„ì„ í˜• ë°ì´í„° ìƒì„±
X_moons, y_moons = make_moons(n_samples=200, noise=0.1, random_state=42)
X_circles, y_circles = make_circles(n_samples=200, noise=0.1, factor=0.5, random_state=42)

# ì»¤ë„ ë¹„êµ
kernels = ['linear', 'poly', 'rbf']

fig, axes = plt.subplots(2, 3, figsize=(15, 10))

for row, (X, y, name) in enumerate([(X_moons, y_moons, 'Moons'),
                                     (X_circles, y_circles, 'Circles')]):
    for col, kernel in enumerate(kernels):
        ax = axes[row, col]

        # SVM í•™ìŠµ
        if kernel == 'poly':
            clf = svm.SVC(kernel=kernel, degree=3, gamma='scale')
        else:
            clf = svm.SVC(kernel=kernel, gamma='scale')
        clf.fit(X, y)

        # ê²°ì • ê²½ê³„
        xlim = [X[:, 0].min() - 0.5, X[:, 0].max() + 0.5]
        ylim = [X[:, 1].min() - 0.5, X[:, 1].max() + 0.5]
        xx, yy = np.meshgrid(np.linspace(xlim[0], xlim[1], 100),
                             np.linspace(ylim[0], ylim[1], 100))

        Z = clf.predict(np.c_[xx.ravel(), yy.ravel()])
        Z = Z.reshape(xx.shape)

        ax.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')
        ax.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='black')
        ax.set_title(f'{name} - {kernel}\nAccuracy: {clf.score(X, y):.3f}')

plt.tight_layout()
plt.show()

3.3 RBF ì»¤ë„ê³¼ gamma íŒŒë¼ë¯¸í„°¶

"""
RBF ì»¤ë„: K(x, y) = exp(-Î³||x - y||Â²)

gamma (Î³):
- ë°ì´í„° í¬ì¸íŠ¸ì˜ ì˜í–¥ ë²”ìœ„ ê²°ì •
- gamma í¼: ê° í¬ì¸íŠ¸ ì˜í–¥ ì¢ìŒ â†’ ë³µì¡í•œ ê²½ê³„, ê³¼ì í•© ìœ„í—˜
- gamma ì‘ìŒ: ê° í¬ì¸íŠ¸ ì˜í–¥ ë„“ìŒ â†’ ë‹¨ìˆœí•œ ê²½ê³„, ê³¼ì†Œì í•© ìœ„í—˜

gamma ì„¤ì •:
- 'scale': 1 / (n_features * X.var()) - ê¸°ë³¸ê°’
- 'auto': 1 / n_features
"""

# gamma íš¨ê³¼
fig, axes = plt.subplots(1, 4, figsize=(20, 5))
gamma_values = [0.1, 1, 10, 100]

X, y = make_moons(n_samples=200, noise=0.1, random_state=42)

for ax, gamma in zip(axes, gamma_values):
    clf = svm.SVC(kernel='rbf', gamma=gamma, C=1)
    clf.fit(X, y)

    xlim = [X[:, 0].min() - 0.5, X[:, 0].max() + 0.5]
    ylim = [X[:, 1].min() - 0.5, X[:, 1].max() + 0.5]
    xx, yy = np.meshgrid(np.linspace(xlim[0], xlim[1], 100),
                         np.linspace(ylim[0], ylim[1], 100))

    Z = clf.predict(np.c_[xx.ravel(), yy.ravel()])
    Z = Z.reshape(xx.shape)

    ax.contourf(xx, yy, Z, alpha=0.3, cmap='coolwarm')
    ax.scatter(X[:, 0], X[:, 1], c=y, cmap='coolwarm', edgecolors='black')
    ax.scatter(clf.support_vectors_[:, 0], clf.support_vectors_[:, 1],
               s=100, facecolors='none', edgecolors='green', linewidths=2)
    ax.set_title(f'gamma = {gamma}\nSVs: {len(clf.support_vectors_)}')

plt.tight_layout()
plt.show()
