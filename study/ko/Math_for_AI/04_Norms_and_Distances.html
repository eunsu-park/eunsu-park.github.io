1. ë²¡í„° ë…¸ë¦„ (Vector Norms)¶

1.1 ë…¸ë¦„ì˜ ì •ì˜¶

ë²¡í„° ê³µê°„ì˜ ë…¸ë¦„ì€ ë‹¤ìŒ ì„¸ ê°€ì§€ ì„±ì§ˆì„ ë§Œì¡±í•˜ëŠ” í•¨ìˆ˜ $\|\cdot\|: \mathbb{R}^n \to \mathbb{R}_+$ì…ë‹ˆë‹¤:

  1. ì–‘ì˜ ì •ë¶€í˜¸ì„±: $\|\mathbf{x}\| \geq 0$ì´ê³ , $\|\mathbf{x}\| = 0 \Leftrightarrow \mathbf{x} = \mathbf{0}$
  2. ë™ì°¨ì„±: $\|\alpha \mathbf{x}\| = |\alpha| \|\mathbf{x}\|$ for all $\alpha \in \mathbb{R}$
  3. ì‚¼ê° ë¶€ë“±ì‹: $\|\mathbf{x} + \mathbf{y}\| \leq \|\mathbf{x}\| + \|\mathbf{y}\|$

1.2 Lp ë…¸ë¦„¶

ì¼ë°˜ì ì¸ $L_p$ ë…¸ë¦„ì€ ë‹¤ìŒê³¼ ê°™ì´ ì •ì˜ë©ë‹ˆë‹¤:

$$\|\mathbf{x}\|_p = \left( \sum_{i=1}^n |x_i|^p \right)^{1/p}$$

1.3 L1 ë…¸ë¦„ (ë§¨í•´íŠ¼ ê±°ë¦¬)¶

$$\|\mathbf{x}\|_1 = \sum_{i=1}^n |x_i|$$

  • ì ˆëŒ“ê°’ì˜ í•©
  • ë§¨í•´íŠ¼ ê±°ë¦¬, íƒì‹œìº¡ ê±°ë¦¬ë¼ê³ ë„ í•¨
  • í¬ì†Œì„±ì„ ìœ ë„í•˜ëŠ” ì •ê·œí™”ì— ì‚¬ìš©

1.4 L2 ë…¸ë¦„ (ìœ í´ë¦¬ë“œ ê±°ë¦¬)¶

$$\|\mathbf{x}\|_2 = \sqrt{\sum_{i=1}^n x_i^2}$$

  • ê°€ì¥ ì¼ë°˜ì ì¸ ë…¸ë¦„
  • ìœ í´ë¦¬ë“œ ê±°ë¦¬
  • ë¯¸ë¶„ ê°€ëŠ¥í•˜ê³  ë‹¤ë£¨ê¸° ì‰¬ì›€

1.5 Lâˆ ë…¸ë¦„ (ìµœëŒ€ ë…¸ë¦„)¶

$$\|\mathbf{x}\|_\infty = \max_i |x_i|$$

  • ê°€ì¥ í° ì ˆëŒ“ê°’
  • $p \to \infty$ì˜ ê·¹í•œìœ¼ë¡œ ì–»ì–´ì§

2. í–‰ë ¬ ë…¸ë¦„ (Matrix Norms)¶

2.1 í”„ë¡œë² ë‹ˆìš°ìŠ¤ ë…¸ë¦„ (Frobenius Norm)¶

$$\|A\|_F = \sqrt{\sum_{i,j} A_{ij}^2} = \sqrt{\text{tr}(A^T A)}$$

  • ëª¨ë“  ì›ì†Œì˜ ì œê³±í•©ì˜ ì œê³±ê·¼
  • ë²¡í„°ì˜ L2 ë…¸ë¦„ì„ í–‰ë ¬ë¡œ í™•ì¥í•œ ê²ƒ

2.2 ìŠ¤í™íŠ¸ëŸ¼ ë…¸ë¦„ (Spectral Norm)¶

$$\|A\|_2 = \sigma_{\max}(A) = \sqrt{\lambda_{\max}(A^T A)}$$

  • ìµœëŒ€ íŠ¹ì´ê°’
  • ìœ ë„ ë…¸ë¦„: $\|A\|_2 = \max_{\|\mathbf{x}\|_2=1} \|A\mathbf{x}\|_2$
  • í–‰ë ¬ì´ ë²¡í„°ë¥¼ ì–¼ë§ˆë‚˜ ëŠ˜ë¦´ ìˆ˜ ìˆëŠ”ì§€ì˜ ì²™ë„

2.3 í•µ ë…¸ë¦„ (Nuclear Norm)¶

$$\|A\|_* = \sum_i \sigma_i(A)$$

  • ëª¨ë“  íŠ¹ì´ê°’ì˜ í•©
  • ì €ê³„ìˆ˜ í–‰ë ¬ ê·¼ì‚¬ì— ì‚¬ìš© (í–‰ë ¬ ì™„ì„± ë¬¸ì œ)
  • L1 ë…¸ë¦„ì˜ í–‰ë ¬ ë²„ì „ (í¬ì†Œì„± ìœ ë„)

3. ê±°ë¦¬ ì¸¡ë„ (Distance Metrics)¶

3.1 ìœ í´ë¦¬ë“œ ê±°ë¦¬ (Euclidean Distance)¶

$$d(\mathbf{x}, \mathbf{y}) = \|\mathbf{x} - \mathbf{y}\|_2 = \sqrt{\sum_{i=1}^n (x_i - y_i)^2}$$

ê°€ì¥ ì¼ë°˜ì ì¸ ê±°ë¦¬ ì¸¡ë„ì…ë‹ˆë‹¤.

3.2 ë§ˆí• ë¼ë…¸ë¹„ìŠ¤ ê±°ë¦¬ (Mahalanobis Distance)¶

$$d_M(\mathbf{x}, \mathbf{y}) = \sqrt{(\mathbf{x} - \mathbf{y})^T \Sigma^{-1} (\mathbf{x} - \mathbf{y})}$$

  • ê³µë¶„ì‚° í–‰ë ¬ $\Sigma$ë¥¼ ê³ ë ¤
  • ë³€ìˆ˜ ê°„ ìƒê´€ê´€ê³„ì™€ ìŠ¤ì¼€ì¼ì„ ë°˜ì˜
  • ì´ìƒì¹˜ íƒì§€ì— ìœ ìš©

3.3 ì½”ì‚¬ì¸ ìœ ì‚¬ë„ (Cosine Similarity)¶

$$\text{sim}(\mathbf{x}, \mathbf{y}) = \frac{\mathbf{x}^T \mathbf{y}}{\|\mathbf{x}\|_2 \|\mathbf{y}\|_2}$$

$$d_{\text{cosine}}(\mathbf{x}, \mathbf{y}) = 1 - \text{sim}(\mathbf{x}, \mathbf{y})$$

  • ë²¡í„°ì˜ ë°©í–¥ë§Œ ë¹„êµ (í¬ê¸° ë¬´ì‹œ)
  • í…ìŠ¤íŠ¸ ìœ ì‚¬ë„, ì„ë² ë”© ë¹„êµì— ì‚¬ìš©

3.4 í•´ë° ê±°ë¦¬ (Hamming Distance)¶

$$d_H(\mathbf{x}, \mathbf{y}) = \sum_{i=1}^n \mathbb{1}[x_i \neq y_i]$$

  • ë‹¤ë¥¸ ì›ì†Œì˜ ê°œìˆ˜
  • ì´ì§„ ë°ì´í„°, ë²”ì£¼í˜• ë°ì´í„°ì— ì‚¬ìš©

4. ì •ê·œí™”ì™€ ë…¸ë¦„ (Regularization)¶

4.1 L1 ì •ê·œí™” (Lasso)¶

ì†ì‹¤ í•¨ìˆ˜ì— L1 ë…¸ë¦„ í˜ë„í‹° ì¶”ê°€:

$$L(\mathbf{w}) = L_{\text{data}}(\mathbf{w}) + \lambda \|\mathbf{w}\|_1$$

  • í¬ì†Œì„± ìœ ë„ (ë§ì€ ê°€ì¤‘ì¹˜ê°€ ì •í™•íˆ 0)
  • íŠ¹ì§• ì„ íƒ íš¨ê³¼

4.2 L2 ì •ê·œí™” (Ridge)¶

ì†ì‹¤ í•¨ìˆ˜ì— L2 ë…¸ë¦„ í˜ë„í‹° ì¶”ê°€:

$$L(\mathbf{w}) = L_{\text{data}}(\mathbf{w}) + \lambda \|\mathbf{w}\|_2^2$$

  • ê³¼ì í•© ë°©ì§€

4.3 Elastic Net¶

$$L(\mathbf{w}) = L_{\text{data}}(\mathbf{w}) + \lambda_1 \|\mathbf{w}\|_1 + \lambda_2 \|\mathbf{w}\|_2^2$$

# ê·¸ë˜ë””ì–¸íŠ¸ í­ë°œ ë°©ì§€
def train_step_with_clipping(model, X, y, optimizer, max_norm=1.0):
    """ê·¸ë˜ë””ì–¸íŠ¸ í´ë¦¬í•‘ì„ ì ìš©í•œ í•™ìŠµ ìŠ¤í…"""
    optimizer.zero_grad()
    output = model(X)
    loss = nn.MSELoss()(output, y)
    loss.backward()

    # ê·¸ë˜ë””ì–¸íŠ¸ ë…¸ë¦„ ê³„ì‚°
    total_norm = 0.0
    for p in model.parameters():
        if p.grad is not None:
            param_norm = p.grad.data.norm(2)
            total_norm += param_norm.item() ** 2
    total_norm = total_norm ** 0.5

    # í´ë¦¬í•‘
    if total_norm > max_norm:
        clip_coef = max_norm / (total_norm + 1e-6)
        for p in model.parameters():
            if p.grad is not None:
                p.grad.data.mul_(clip_coef)
        clipped_norm = max_norm
    else:
        clipped_norm = total_norm

    optimizer.step()
    return loss.item(), total_norm, clipped_norm

# í…ŒìŠ¤íŠ¸

    model, X, y, optimizer, max_norm=1.0
)

print("ê·¸ë˜ë””ì–¸íŠ¸ í´ë¦¬í•‘:")
print(f"ì›ë˜ ê·¸ë˜ë””ì–¸íŠ¸ ë…¸ë¦„: {orig_norm:.4f}")
print(f"í´ë¦¬í•‘ í›„ ë…¸ë¦„:       {clipped_norm:.4f}")
print(f"í´ë¦¬í•‘ ë°œìƒ:          {orig_norm > 1.0}")
