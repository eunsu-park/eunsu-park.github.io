1. í¸ë¯¸ë¶„ (Partial Derivatives)¶

1.1 í¸ë¯¸ë¶„ì˜ ì •ì˜¶

ë‹¤ë³€ìˆ˜ í•¨ìˆ˜ $f(x_1, x_2, \ldots, x_n)$ì—ì„œ $x_i$ì— ëŒ€í•œ í¸ë¯¸ë¶„ì€ ë‹¤ë¥¸ ë³€ìˆ˜ë¥¼ ìƒìˆ˜ë¡œ ê³ ì •í•˜ê³  $x_i$ë§Œ ë³€í™”ì‹œí‚¬ ë•Œì˜ ë³€í™”ìœ¨ì…ë‹ˆë‹¤:

$$\frac{\partial f}{\partial x_i} = \lim_{h \to 0} \frac{f(x_1, \ldots, x_i + h, \ldots, x_n) - f(x_1, \ldots, x_i, \ldots, x_n)}{h}$$

1.2 í¸ë¯¸ë¶„ ê³„ì‚° ì˜ˆì œ¶

ì˜ˆì œ 1: $f(x, y) = x^2 + 3xy + y^2$

$$\frac{\partial f}{\partial x} = 2x + 3y$$

$$\frac{\partial f}{\partial y} = 3x + 2y$$

ì˜ˆì œ 2: $f(x, y) = e^{x^2 + y^2}$

$$\frac{\partial f}{\partial x} = 2x e^{x^2 + y^2}$$

$$\frac{\partial f}{\partial y} = 2y e^{x^2 + y^2}$$

    """2D ê·¸ë˜ë””ì–¸íŠ¸ í•˜ê°•ë²•"""

3. ë°©í–¥ ë„í•¨ìˆ˜ (Directional Derivative)¶

4. ë‹¤ë³€ìˆ˜ ì²´ì¸ ë£° (Multivariate Chain Rule)¶

4.1 ì²´ì¸ ë£°ì˜ í˜•íƒœ¶

$z = f(x, y)$ì´ê³  $x = x(t)$, $y = y(t)$ì¼ ë•Œ:

$$\frac{dz}{dt} = \frac{\partial f}{\partial x} \frac{dx}{dt} + \frac{\partial f}{\partial y} \frac{dy}{dt}$$

ì¼ë°˜ì ìœ¼ë¡œ:

$$\frac{dz}{dt} = \nabla f \cdot \frac{d\mathbf{x}}{dt}$$

4.2 ì²´ì¸ ë£° ì˜ˆì œ¶

def f_example(x, y):
    return np.exp(-(x**2 + y**2)) * np.sin(x) * np.cos(y)

def grad_f_example(x, y):
    """ìˆ˜ì¹˜ ê·¸ë˜ë””ì–¸íŠ¸"""
    h = 1e-5
    df_dx = (f_example(x+h, y) - f_example(x-h, y)) / (2*h)
    df_dy = (f_example(x, y+h) - f_example(x, y-h)) / (2*h)
    return np.array([df_dx, df_dy])

def hessian_f_example(x, y):
    """ìˆ˜ì¹˜ í—¤ì‹œì•ˆ"""
    h = 1e-5
    H = np.zeros((2, 2))
    grad_base = grad_f_example(x, y)

    # H[0,0] = âˆ‚Â²f/âˆ‚xÂ²
    grad_x_plus = grad_f_example(x+h, y)
    H[0, 0] = (grad_x_plus[0] - grad_base[0]) / h

    # H[1,1] = âˆ‚Â²f/âˆ‚yÂ²
    grad_y_plus = grad_f_example(x, y+h)
    H[1, 1] = (grad_y_plus[1] - grad_base[1]) / h

    # H[0,1] = H[1,0] = âˆ‚Â²f/âˆ‚xâˆ‚y
    H[0, 1] = (grad_x_plus[1] - grad_base[1]) / h
    H[1, 0] = H[0, 1]

    return H

            break

    return np.array(path)
