2. êµì°¨ ì—”íŠ¸ë¡œí”¼ (Cross-Entropy)¶

2.1 ì •ì˜¶

êµì°¨ ì—”íŠ¸ë¡œí”¼ëŠ” ë¶„í¬ $P$ë¡œë¶€í„° ìƒ˜í”Œë§ëœ ë°ì´í„°ë¥¼ ë¶„í¬ $Q$ë¡œ ë¶€í˜¸í™”í•  ë•Œ í•„ìš”í•œ í‰ê·  ë¹„íŠ¸ ìˆ˜ì…ë‹ˆë‹¤:

$$H(P, Q) = -\sum_{x} P(x) \log Q(x) = \mathbb{E}_{P(x)}[-\log Q(x)]$$

í•´ì„: - $P$: ì‹¤ì œ ë¶„í¬ (ë°ì´í„°) - $Q$: ëª¨ë¸ ë¶„í¬ (ì˜ˆì¸¡) - $H(P, Q)$: $Q$ë¡œ $P$ë¥¼ ê·¼ì‚¬í•  ë•Œì˜ "ë¹„ìš©"

ì„±ì§ˆ: 1. $H(P, Q) \geq H(P)$ (ë“±í˜¸ëŠ” $P = Q$ì¼ ë•Œ) 2. ë¹„ëŒ€ì¹­: $H(P, Q) \neq H(Q, P)$ (ì¼ë°˜ì ìœ¼ë¡œ)

2.2 ì´ì§„ êµì°¨ ì—”íŠ¸ë¡œí”¼ (Binary Cross-Entropy)¶

ë¡œì§€ìŠ¤í‹± íšŒê·€ì—ì„œ ì‚¬ìš©:

$$H(y, \hat{y}) = -[y \log \hat{y} + (1-y) \log(1-\hat{y})]$$

  • $y \in \{0, 1\}$: ì‹¤ì œ ë ˆì´ë¸”
  • $\hat{y} \in [0, 1]$: ì˜ˆì¸¡ í™•ë¥ 

nê°œ ìƒ˜í”Œì˜ í‰ê· :

$$\mathcal{L}_{\text{BCE}} = -\frac{1}{n}\sum_{i=1}^n [y_i \log \hat{y}_i + (1-y_i) \log(1-\hat{y}_i)]$$

2.3 ì¹´í…Œê³ ë¦¬ì»¬ êµì°¨ ì—”íŠ¸ë¡œí”¼ (Categorical Cross-Entropy)¶

ë‹¤ì¤‘ ë¶„ë¥˜ì—ì„œ ì‚¬ìš©:

$$H(P, Q) = -\sum_{k=1}^K P_k \log Q_k$$

One-hot ì¸ì½”ë”©ëœ ê²½ìš° ($P_k = \delta_{k,c}$):

$$\mathcal{L}_{\text{CCE}} = -\log Q_c$$

ì—¬ê¸°ì„œ $c$ëŠ” ì •ë‹µ í´ë˜ìŠ¤.

3. KL ë°œì‚° (Kullback-Leibler Divergence)¶

3.1 ì •ì˜¶

KL ë°œì‚°ì€ ë‘ ë¶„í¬ $P$ì™€ $Q$ ì‚¬ì´ì˜ "ê±°ë¦¬"ë¥¼ ì¸¡ì •í•©ë‹ˆë‹¤:

$$D_{\text{KL}}(P \| Q) = \sum_{x} P(x) \log \frac{P(x)}{Q(x)} = \mathbb{E}_{P(x)}\left[\log \frac{P(x)}{Q(x)}\right]$$

ì—°ì† ë¶„í¬:

$$D_{\text{KL}}(P \| Q) = \int p(x) \log \frac{p(x)}{q(x)} dx$$

3.2 KL ë°œì‚°ì˜ ì„±ì§ˆ¶

  1. ë¹„ìŒì„±: $D_{\text{KL}}(P \| Q) \geq 0$
  2. ë“±í˜¸ ì¡°ê±´: $D_{\text{KL}}(P \| Q) = 0 \Leftrightarrow P = Q$ (ê±°ì˜ ëª¨ë“  ê³³ì—ì„œ)
  3. ë¹„ëŒ€ì¹­: $D_{\text{KL}}(P \| Q) \neq D_{\text{KL}}(Q \| P)$ â†’ ê±°ë¦¬ í•¨ìˆ˜ê°€ ì•„ë‹˜!
  4. êµì°¨ ì—”íŠ¸ë¡œí”¼ì™€ì˜ ê´€ê³„: $$D_{\text{KL}}(P \| Q) = H(P, Q) - H(P)$$

3.3 ë¹„ìŒì„± ì¦ëª… (ê¹ìŠ¤ ë¶€ë“±ì‹)¶

Jensen ë¶€ë“±ì‹ ì‚¬ìš©: $-\log$ëŠ” ë³¼ë¡ í•¨ìˆ˜ì´ë¯€ë¡œ,

$$-\mathbb{E}[\log X] \geq -\log \mathbb{E}[X]$$

ì ìš©:

$$D_{\text{KL}}(P \| Q) = \mathbb{E}_{P(x)}\left[\log \frac{P(x)}{Q(x)}\right]$$

$$= -\mathbb{E}_{P(x)}\left[\log \frac{Q(x)}{P(x)}\right]$$

$$\geq -\log \mathbb{E}_{P(x)}\left[\frac{Q(x)}{P(x)}\right]$$

$$= -\log \sum_{x} P(x) \frac{Q(x)}{P(x)}$$

$$= -\log \sum_{x} Q(x) = -\log 1 = 0$$

3.4 Forward KL vs Reverse KL¶

Forward KL: $D_{\text{KL}}(P \| Q)$ - $Q$ê°€ $P$ë¥¼ ì»¤ë²„í•˜ë„ë¡ ê°•ì œ (mode-covering) - $P(x) > 0$ì´ë©´ $Q(x) > 0$ì´ì–´ì•¼ í•¨

Reverse KL: $D_{\text{KL}}(Q \| P)$ - $Q$ê°€ $P$ì˜ ëª¨ë“œë¥¼ ì„ íƒ (mode-seeking) - $Q$ëŠ” $P$ì˜ í•œ ëª¨ë“œì—ë§Œ ì§‘ì¤‘í•  ìˆ˜ ìˆìŒ

import numpy as np
import matplotlib.pyplot as plt
from scipy import stats

# ë‘ ê°œì˜ ê°€ìš°ì‹œì•ˆ í˜¼í•©ì„ íƒ€ê²Ÿìœ¼ë¡œ
np.random.seed(42)
x = np.linspace(-5, 10, 1000)

# íƒ€ê²Ÿ: ë‘ ê°œì˜ ê°€ìš°ì‹œì•ˆ í˜¼í•©
p = 0.5 * stats.norm.pdf(x, 0, 1) + 0.5 * stats.norm.pdf(x, 5, 1)
p = p / np.trapz(p, x)  # ì •ê·œí™”

# Forward KLë¡œ ê·¼ì‚¬ (mode-covering)
def kl_divergence_forward(mu, sigma, x, p_true):
    q = stats.norm.pdf(x, mu, sigma)
    q = q / np.trapz(q, x)
    # Forward KL: E_p[log(p/q)]
    kl = np.trapz(p_true * np.log((p_true + 1e-10) / (q + 1e-10)), x)
    return kl

# Reverse KLë¡œ ê·¼ì‚¬ (mode-seeking)
def kl_divergence_reverse(mu, sigma, x, p_true):
    q = stats.norm.pdf(x, mu, sigma)
    q = q / np.trapz(q, x)
    # Reverse KL: E_q[log(q/p)]
    kl = np.trapz(q * np.log((q + 1e-10) / (p_true + 1e-10)), x)
    return kl

# ê·¸ë¦¬ë“œ ì„œì¹˜ë¡œ ìµœì í™” (ì‹¤ì œë¡œëŠ” ê²½ì‚¬í•˜ê°•ë²• ì‚¬ìš©)
mu_range = np.linspace(-2, 7, 50)
sigma_range = np.linspace(0.5, 3, 30)

best_forward_kl = float('inf')
best_reverse_kl = float('inf')
best_forward_params = None
best_reverse_params = None

for mu in mu_range:
    for sigma in sigma_range:
        fkl = kl_divergence_forward(mu, sigma, x, p)
        rkl = kl_divergence_reverse(mu, sigma, x, p)

        if fkl < best_forward_kl:
            best_forward_kl = fkl
            best_forward_params = (mu, sigma)

        if rkl < best_reverse_kl:
            best_reverse_kl = rkl
            best_reverse_params = (mu, sigma)

# ìµœì  ë¶„í¬
q_forward = stats.norm.pdf(x, *best_forward_params)
q_forward = q_forward / np.trapz(q_forward, x)

q_reverse = stats.norm.pdf(x, *best_reverse_params)

4. ìƒí˜¸ ì •ë³´ëŸ‰ (Mutual Information)¶

4.1 ì •ì˜¶

ë‘ í™•ë¥  ë³€ìˆ˜ $X$ì™€ $Y$ ì‚¬ì´ì˜ ìƒí˜¸ ì •ë³´ëŸ‰:

$$I(X; Y) = \sum_{x, y} P(x, y) \log \frac{P(x, y)}{P(x)P(y)}$$

$$= D_{\text{KL}}(P(X, Y) \| P(X)P(Y))$$

ë‹¤ë¥¸ í‘œí˜„:

$$I(X; Y) = H(X) - H(X|Y) = H(Y) - H(Y|X)$$

$$= H(X) + H(Y) - H(X, Y)$$

4.2 ì¡°ê±´ë¶€ ì—”íŠ¸ë¡œí”¼¶

$Y$ê°€ ì£¼ì–´ì¡Œì„ ë•Œ $X$ì˜ ì¡°ê±´ë¶€ ì—”íŠ¸ë¡œí”¼:

$$H(X|Y) = \sum_{y} P(y) H(X|Y=y)$$

$$= -\sum_{x, y} P(x, y) \log P(x|y)$$

ì²´ì¸ ê·œì¹™:

$$H(X, Y) = H(X) + H(Y|X) = H(Y) + H(X|Y)$$

4.3 ìƒí˜¸ ì •ë³´ëŸ‰ì˜ ì„±ì§ˆ¶

  1. ë¹„ìŒì„±: $I(X; Y) \geq 0$
  2. ëŒ€ì¹­ì„±: $I(X; Y) = I(Y; X)$
  3. ë…ë¦½ì„±: $I(X; Y) = 0 \Leftrightarrow X \perp Y$
  4. ìƒí•œ: $I(X; Y) \leq \min(H(X), H(Y))$

ì§ê´€: - $I(X; Y)$: $X$ë¥¼ ì•Œë©´ $Y$ì— ëŒ€í•œ ë¶ˆí™•ì‹¤ì„±ì´ ì–¼ë§ˆë‚˜ ì¤„ì–´ë“œëŠ”ê°€? - $= H(Y) - H(Y|X)$: $Y$ì˜ ì—”íŠ¸ë¡œí”¼ì—ì„œ $X$ë¥¼ ì•Œì•˜ì„ ë•Œì˜ ì¡°ê±´ë¶€ ì—”íŠ¸ë¡œí”¼ë¥¼ ëº€ ê²ƒ

4.4 ì‘ìš©: íŠ¹ì§• ì„ íƒ¶

5. ì˜Œì„¼ ë¶€ë“±ì‹ (Jensen's Inequality)¶

5.1 ì •ì˜¶

í•¨ìˆ˜ $f$ê°€ ë³¼ë¡ í•¨ìˆ˜(convex)ì´ë©´:

$$f(\mathbb{E}[X]) \leq \mathbb{E}[f(X)]$$

í•¨ìˆ˜ $f$ê°€ ì˜¤ëª© í•¨ìˆ˜(concave)ì´ë©´:

$$f(\mathbb{E}[X]) \geq \mathbb{E}[f(X)]$$

ì´ì‚° í˜•íƒœ:

$$f\left(\sum_i \lambda_i x_i\right) \leq \sum_i \lambda_i f(x_i)$$

ì—¬ê¸°ì„œ $\lambda_i \geq 0, \sum_i \lambda_i = 1$.

5.2 ë³¼ë¡ í•¨ìˆ˜ì˜ ì˜ˆ¶

  • $f(x) = x^2$
  • $f(x) = e^x$
  • $f(x) = -\log x$ (for $x > 0$)

5.3 ì‘ìš©: KL ë°œì‚°ì˜ ë¹„ìŒì„±¶

$$D_{\text{KL}}(P \| Q) = \mathbb{E}_{P}\left[\log \frac{P(x)}{Q(x)}\right]$$

$$= -\mathbb{E}_{P}\left[\log \frac{Q(x)}{P(x)}\right]$$

$-\log$ëŠ” ë³¼ë¡ í•¨ìˆ˜ì´ë¯€ë¡œ Jensen ë¶€ë“±ì‹:

$$-\mathbb{E}_{P}\left[\log \frac{Q(x)}{P(x)}\right] \geq -\log \mathbb{E}_{P}\left[\frac{Q(x)}{P(x)}\right]$$

$$= -\log \sum_{x} P(x) \frac{Q(x)}{P(x)} = -\log \sum_{x} Q(x) = 0$$

5.4 ì‘ìš©: ELBO ìœ ë„ (VAE)¶

VAEì—ì„œ Evidence Lower BOund (ELBO):

$$\log P(x) = \log \int P(x, z) dz = \log \int Q(z) \frac{P(x, z)}{Q(z)} dz$$

Jensen ($\log$ëŠ” ì˜¤ëª©):

$$\geq \int Q(z) \log \frac{P(x, z)}{Q(z)} dz$$

$$= \mathbb{E}_{Q(z)}[\log P(x, z)] - \mathbb{E}_{Q(z)}[\log Q(z)]$$

$$= \mathbb{E}_{Q(z)}[\log P(x|z)] - D_{\text{KL}}(Q(z) \| P(z))$$

ì´ê²ƒì´ VAEì˜ ì†ì‹¤ í•¨ìˆ˜ì…ë‹ˆë‹¤!

6. ë¨¸ì‹ ëŸ¬ë‹ì—ì„œì˜ ì •ë³´ ì´ë¡ ¶

6.1 êµì°¨ ì—”íŠ¸ë¡œí”¼ ì†ì‹¤ = MLE¶

ë¶„ë¥˜ ë¬¸ì œì—ì„œ êµì°¨ ì—”íŠ¸ë¡œí”¼ ìµœì†Œí™” = ìŒì˜ ë¡œê·¸ ìš°ë„ ìµœì†Œí™”:

$$\min_{\theta} H(P_{\text{data}}, P_{\theta}) = \min_{\theta} -\mathbb{E}_{(x,y) \sim P_{\text{data}}}[\log P_{\theta}(y|x)]$$

ì´ëŠ” MLEì™€ ë™ì¼í•©ë‹ˆë‹¤!

6.2 VAEì˜ ELBO¶

Variational Autoencoderì˜ ëª©ì  í•¨ìˆ˜:

$$\mathcal{L}_{\text{VAE}} = \mathbb{E}_{q_\phi(z|x)}[\log p_\theta(x|z)] - D_{\text{KL}}(q_\phi(z|x) \| p(z))$$

  • ì²« ë²ˆì§¸ í•­: ì¬êµ¬ì„± ì†ì‹¤ (reconstruction loss)
  • ë‘ ë²ˆì§¸ í•­: KL ì •ê·œí™” (KL regularization)
import torch
import torch.nn as nn
import torch.nn.functional as F

class VAE(nn.Module):
    def __init__(self, input_dim=784, hidden_dim=400, latent_dim=20):
        super(VAE, self).__init__()

        # Encoder
        self.fc1 = nn.Linear(input_dim, hidden_dim)
        self.fc_mu = nn.Linear(hidden_dim, latent_dim)
        self.fc_logvar = nn.Linear(hidden_dim, latent_dim)

        # Decoder
        self.fc3 = nn.Linear(latent_dim, hidden_dim)
        self.fc4 = nn.Linear(hidden_dim, input_dim)

    def encode(self, x):
        h = F.relu(self.fc1(x))
        return self.fc_mu(h), self.fc_logvar(h)

    def reparameterize(self, mu, logvar):
        std = torch.exp(0.5 * logvar)
        eps = torch.randn_like(std)
        return mu + eps * std

    def decode(self, z):
        h = F.relu(self.fc3(z))
        return torch.sigmoid(self.fc4(h))

    def forward(self, x):
        mu, logvar = self.encode(x)
        z = self.reparameterize(mu, logvar)
        return self.decode(z), mu, logvar

def vae_loss(recon_x, x, mu, logvar):
    # ì¬êµ¬ì„± ì†ì‹¤ (Binary Cross-Entropy)
    BCE = F.binary_cross_entropy(recon_x, x, reduction='sum')

    # KL ë°œì‚°: D_KL(q(z|x) || p(z))
    # p(z) = N(0, I)ì´ë¯€ë¡œ í•´ì„ì ìœ¼ë¡œ ê³„ì‚° ê°€ëŠ¥
    # KL = -0.5 * sum(1 + log(sigma^2) - mu^2 - sigma^2)
    KLD = -0.5 * torch.sum(1 + logvar - mu.pow(2) - logvar.exp())

    return BCE + KLD, BCE, KLD

# ê°„ë‹¨í•œ í…ŒìŠ¤íŠ¸
model = VAE(input_dim=784, latent_dim=20)
x = torch.randn(32, 784)  # ë°°ì¹˜ í¬ê¸° 32

recon_x, mu, logvar = model(x)
loss, bce, kld = vae_loss(recon_x, x, mu, logvar)

print(f"Total Loss: {loss.item():.2f}")
print(f"Reconstruction Loss (BCE): {bce.item():.2f}")
print(f"KL Divergence: {kld.item():.2f}")
print(f"\nELBO = -Loss = {-loss.item():.2f}")

6.3 GANì˜ JS ë°œì‚°¶

Generative Adversarial Networkì˜ ì›ë˜ ëª©ì  í•¨ìˆ˜ëŠ” Jensen-Shannon (JS) ë°œì‚°ê³¼ ê´€ë ¨:

$$D_{\text{JS}}(P \| Q) = \frac{1}{2}D_{\text{KL}}(P \| M) + \frac{1}{2}D_{\text{KL}}(Q \| M)$$

ì—¬ê¸°ì„œ $M = \frac{1}{2}(P + Q)$.

GANì˜ ìµœì  íŒë³„ê¸°ëŠ” JS ë°œì‚°ì„ ì¶”ì •í•©ë‹ˆë‹¤.

6.4 ì •ë³´ ë³‘ëª© (Information Bottleneck)¶

ì •ë³´ ë³‘ëª© ì´ë¡ ì€ ì…ë ¥ $X$ì™€ íƒ€ê²Ÿ $Y$ ì‚¬ì´ì— ì••ì¶•ëœ í‘œí˜„ $Z$ë¥¼ ì°¾ìŠµë‹ˆë‹¤:

$$\min_{p(z|x)} I(X; Z) - \beta I(Z; Y)$$

  • $I(X; Z)$: ìµœì†Œí™” â†’ ì••ì¶•
  • $I(Z; Y)$: ìµœëŒ€í™” â†’ ì •ë³´ ë³´ì¡´
  • $\beta$: trade-off íŒŒë¼ë¯¸í„°

ì´ëŠ” ë”¥ëŸ¬ë‹ì˜ ì´ë¡ ì  ì´í•´ì— ì‚¬ìš©ë©ë‹ˆë‹¤.

import numpy as np
import matplotlib.pyplot as plt

# JS ë°œì‚° ì‹œê°í™”
def js_divergence(p, q):
    """Jensen-Shannon divergence"""
    m = 0.5 * (p + q)
    kl_pm = np.sum(p * np.log((p + 1e-10) / (m + 1e-10)))
    kl_qm = np.sum(q * np.log((q + 1e-10) / (m + 1e-10)))
    return 0.5 * kl_pm + 0.5 * kl_qm

ì—°ìŠµ ë¬¸ì œ¶

ë¬¸ì œ 1: ìµœëŒ€ ì—”íŠ¸ë¡œí”¼ ì¦ëª…¶

ë¼ê·¸ë‘ì£¼ ìŠ¹ìˆ˜ë²•ì„ ì‚¬ìš©í•˜ì—¬, ì œì•½ ì¡°ê±´ $\sum_{i=1}^n p_i = 1$í•˜ì—ì„œ ì—”íŠ¸ë¡œí”¼ $H = -\sum_{i=1}^n p_i \log p_i$ë¥¼ ìµœëŒ€í™”í•˜ëŠ” ë¶„í¬ê°€ ê· ë“± ë¶„í¬ $p_i = 1/n$ì„ì„ ì¦ëª…í•˜ì‹œì˜¤.

íŒíŠ¸: - ë¼ê·¸ë‘ì§€ì•ˆ: $L = -\sum_i p_i \log p_i - \lambda(\sum_i p_i - 1)$ - $\frac{\partial L}{\partial p_i} = 0$ í’€ê¸°

ë¬¸ì œ 2: ì¡°ê±´ë¶€ ì—”íŠ¸ë¡œí”¼ì™€ ìƒí˜¸ ì •ë³´ëŸ‰¶

ë‹¤ìŒì„ ì¦ëª…í•˜ì‹œì˜¤:

(a) $H(X, Y) = H(X) + H(Y|X)$ (ì²´ì¸ ê·œì¹™)

(b) $I(X; Y) = H(X) + H(Y) - H(X, Y)$

(c) $I(X; Y) \leq \min(H(X), H(Y))$

(d) $I(X; Y) = 0 \Leftrightarrow X \perp Y$

ë¬¸ì œ 3: KL ë°œì‚° ê³„ì‚°¶

ë‘ ê°€ìš°ì‹œì•ˆ ë¶„í¬ $P = \mathcal{N}(\mu_1, \sigma_1^2)$ì™€ $Q = \mathcal{N}(\mu_2, \sigma_2^2)$ ì‚¬ì´ì˜ KL ë°œì‚°ì„ í•´ì„ì ìœ¼ë¡œ ê³„ì‚°í•˜ì‹œì˜¤.
