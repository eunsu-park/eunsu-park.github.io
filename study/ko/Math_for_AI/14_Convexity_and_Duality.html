$$ \mathcal{L}(w, b, \alpha) = \frac{1}{2} \|w\|^2 - \sum_{i=1}^{n} \alpha_i [y_i(w^T x_i + b) - 1] $$

ì—¬ê¸°ì„œ $\alpha_i \geq 0$ì€ ë¼ê·¸ë‘ì£¼ ìŠ¹ìˆ˜ì…ë‹ˆë‹¤.

3.3 KKT ì¡°ê±´¶

1. ì •ìƒì„±(Stationarity): $$ \nabla_w \mathcal{L} = w - \sum_{i=1}^{n} \alpha_i y_i x_i = 0 \quad \Rightarrow \quad w = \sum_{i=1}^{n} \alpha_i y_i x_i $$

$$ \frac{\partial \mathcal{L}}{\partial b} = -\sum_{i=1}^{n} \alpha_i y_i = 0 \quad \Rightarrow \quad \sum_{i=1}^{n} \alpha_i y_i = 0 $$

2. ìƒë³´ ìŠ¬ë™ì„±(Complementary Slackness): $$ \alpha_i [y_i(w^T x_i + b) - 1] = 0 $$

3. ì›ì´ˆ/ìŒëŒ€ ì‹¤í–‰ ê°€ëŠ¥ì„±: $y_i(w^T x_i + b) \geq 1$, $\alpha_i \geq 0$

3.4 ìŒëŒ€ ë¬¸ì œ ìœ ë„¶

$w = \sum \alpha_i y_i x_i$ë¥¼ ë¼ê·¸ë‘ì§€ì•ˆì— ëŒ€ì…:

$$ \begin{align} \mathcal{L}(w, b, \alpha) &= \frac{1}{2} \left\|\sum_{i} \alpha_i y_i x_i\right\|^2 - \sum_{i,j} \alpha_i \alpha_j y_i y_j (x_i^T x_j) - b\sum_i \alpha_i y_i + \sum_i \alpha_i \\ &= -\frac{1}{2} \sum_{i,j} \alpha_i \alpha_j y_i y_j (x_i^T x_j) + \sum_i \alpha_i \end{align} $$

ìŒëŒ€ ë¬¸ì œ: $$ \begin{align} \max_{\alpha} \quad & \sum_{i=1}^{n} \alpha_i - \frac{1}{2} \sum_{i,j=1}^{n} \alpha_i \alpha_j y_i y_j (x_i^T x_j) \\ \text{s.t.} \quad & \sum_{i=1}^{n} \alpha_i y_i = 0 \\ & \alpha_i \geq 0, \quad i = 1, \ldots, n \end{align} $$

5.1 ì •ì˜¶

í•¨ìˆ˜ $f$ì˜ ê·¼ìœ„ ì—°ì‚°ì: $$ \text{prox}_f(v) = \arg\min_{x} \left( f(x) + \frac{1}{2}\|x - v\|^2 \right) $$

$f(x) = \lambda \|x\|_1$ì¼ ë•Œ: $$ [\text{prox}_{\lambda \|\cdot\|_1}(v)]_i = \text{sign}(v_i) \max(|v_i| - \lambda, 0) $$

def soft_threshold(v, lambda_param):

    return np.sign(v) * np.maximum(np.abs(v) - lambda_param, 0)

5.3 ê·¼ìœ„ ê²½ì‚¬ í•˜ê°•ë²• (Proximal Gradient Descent)¶

ë¬¸ì œ: $\min f(x) + g(x)$, ì—¬ê¸°ì„œ $f$ëŠ” ë§¤ë„ëŸ½ê³  $g$ëŠ” ë¹„ë§¤ë„ëŸ¬ì›€ (ì˜ˆ: L1)

# LASSO ë¬¸ì œ: min 1/2 ||Ax - b||^2 + Î»||x||_1
def lasso_proximal_gradient(A, b, lambda_param, max_iter=1000, alpha=0.01):
    """ê·¼ìœ„ ê²½ì‚¬ í•˜ê°•ë²•ìœ¼ë¡œ LASSO í’€ê¸°"""
    n = A.shape[1]
    x = np.zeros(n)
    losses = []

    for k in range(max_iter):
        # ë§¤ë„ëŸ¬ìš´ ë¶€ë¶„ì˜ ê·¸ë˜ë””ì–¸íŠ¸: âˆ‡(1/2 ||Ax - b||^2) = A^T(Ax - b)
        residual = A @ x - b
        grad_f = A.T @ residual

        # ê²½ì‚¬ í•˜ê°•
        x_temp = x - alpha * grad_f

        # ê·¼ìœ„ ì—°ì‚° (L1)

        # ì†ì‹¤ ê³„ì‚°

        losses.append(loss)

        if k > 0 and abs(losses[-1] - losses[-2]) < 1e-6:
            break

    return x, losses

6.2 ADMM (Alternating Direction Method of Multipliers)¶

ë¬¸ì œ: $\min f(x) + g(z)$ s.t. $Ax + Bz = c$

ADMM ì•Œê³ ë¦¬ì¦˜: 1. $x_{k+1} = \arg\min_x \mathcal{L}_\rho(x, z_k, \lambda_k)$ 2. $z_{k+1} = \arg\min_z \mathcal{L}_\rho(x_{k+1}, z, \lambda_k)$ 3. $\lambda_{k+1} = \lambda_k + \rho(Ax_{k+1} + Bz_{k+1} - c)$

ì—¬ê¸°ì„œ $\mathcal{L}_\rho$ëŠ” ì¦ê°• ë¼ê·¸ë‘ì§€ì•ˆì…ë‹ˆë‹¤.

# ADMMìœ¼ë¡œ LASSO í’€ê¸°
def lasso_admm(A, b, lambda_param, rho=1.0, max_iter=100):
    """ADMMìœ¼ë¡œ LASSO: min 1/2||Ax-b||^2 + Î»||z||_1, s.t. x = z"""
    n, p = A.shape
    x = np.zeros(p)
    z = np.zeros(p)
    u = np.zeros(p)  # ìŠ¤ì¼€ì¼ëœ ìŒëŒ€ ë³€ìˆ˜

    # ì‚¬ì „ ê³„ì‚°
    AtA = A.T @ A
    Atb = A.T @ b
    I = np.eye(p)

    losses = []

    for k in range(max_iter):
        # x ì—…ë°ì´íŠ¸: (A^T A + ÏI)x = A^T b + Ï(z - u)
        x = np.linalg.solve(AtA + rho * I, Atb + rho * (z - u))

        # z ì—…ë°ì´íŠ¸: soft thresholding
        z_old = z.copy()
        z = soft_threshold(x + u, lambda_param / rho)

        # u ì—…ë°ì´íŠ¸ (ìŒëŒ€ ë³€ìˆ˜)
        u = u + x - z

        # ì†ì‹¤
        loss = 0.5 * np.linalg.norm(A @ x - b)**2 + lambda_param * np.linalg.norm(z, 1)
        losses.append(loss)

        # ìˆ˜ë ´ í™•ì¸
        if np.linalg.norm(z - z_old) < 1e-4:
            break

    return x, z, losses
