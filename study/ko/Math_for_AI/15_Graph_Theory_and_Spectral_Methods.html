2. ê·¸ë˜í”„ ë¼í”Œë¼ì‹œì•ˆ (Graph Laplacian)¶

2.1 ë¼í”Œë¼ì‹œì•ˆì˜ ì •ì˜¶

ë¹„ì •ê·œí™” ë¼í”Œë¼ì‹œì•ˆ:

$$L = D - A$$

ì„±ì§ˆ: - ëŒ€ì¹­: $L = L^T$ - ì–‘ë°˜ì •ì¹˜(positive semidefinite): $\mathbf{x}^T L \mathbf{x} \geq 0$ - $L \mathbf{1} = \mathbf{0}$ (ëª¨ë“  ì›ì†Œê°€ 1ì¸ ë²¡í„°ëŠ” ê³ ìœ ê°’ 0ì˜ ê³ ìœ ë²¡í„°)

2.2 ë¼í”Œë¼ì‹œì•ˆì˜ ì´ì°¨ í˜•ì‹¶

$$\mathbf{x}^T L \mathbf{x} = \mathbf{x}^T(D - A)\mathbf{x} = \sum_{i} d_i x_i^2 - \sum_{i,j} A_{ij} x_i x_j$$

ë¬´ë°©í–¥ ê·¸ë˜í”„ì˜ ê²½ìš°:

$$\mathbf{x}^T L \mathbf{x} = \frac{1}{2} \sum_{i,j} A_{ij}(x_i - x_j)^2$$

ì´ëŠ” ì¸ì ‘í•œ ì •ì  ê°„ì˜ ì°¨ì´ë¥¼ ì¸¡ì •í•˜ëŠ” smoothness ì²™ë„ì…ë‹ˆë‹¤.

def compute_laplacian(A):
    """ê·¸ë˜í”„ ë¼í”Œë¼ì‹œì•ˆ ê³„ì‚°"""
    D, _ = compute_degree_matrix(A)
    L = D - A
    return L

2.3 ì •ê·œí™” ë¼í”Œë¼ì‹œì•ˆ¶

ëŒ€ì¹­ ì •ê·œí™” ë¼í”Œë¼ì‹œì•ˆ:

$$L_{\text{sym}} = D^{-1/2} L D^{-1/2} = I - D^{-1/2} A D^{-1/2}$$

ëœë¤ ì›Œí¬ ì •ê·œí™” ë¼í”Œë¼ì‹œì•ˆ:

$$L_{\text{rw}} = D^{-1} L = I - D^{-1} A$$

ì •ê·œí™” ë¼í”Œë¼ì‹œì•ˆì˜ ê³ ìœ ê°’ì€ $[0, 2]$ ë²”ìœ„ì— ìˆìŠµë‹ˆë‹¤.

def compute_normalized_laplacian(A):
    """ì •ê·œí™” ë¼í”Œë¼ì‹œì•ˆ ê³„ì‚°"""
    D, degrees = compute_degree_matrix(A)

    # D^{-1/2} ê³„ì‚°
    D_inv_sqrt = np.diag(1.0 / np.sqrt(degrees))

    # L_sym = D^{-1/2} L D^{-1/2}
    L = compute_laplacian(A)
    L_sym = D_inv_sqrt @ L @ D_inv_sqrt

    # L_rw = D^{-1} L
    D_inv = np.diag(1.0 / degrees)
    L_rw = D_inv @ L

    return L_sym, L_rw

ê·¸ë˜í”„ë¥¼ ë‘ ë¶€ë¶„ $S$ì™€ $\bar{S}$ë¡œ ë¶„í• í•  ë•Œ, ì ˆë‹¨ ë¹„ìš©:

3.2 Fiedler ë²¡í„°ì™€ ìŠ¤í™íŠ¸ëŸ¼ ë°©ë²•¶

Ncut ë¬¸ì œëŠ” NP-hardì´ì§€ë§Œ, ë¼í”Œë¼ì‹œì•ˆì˜ ë‘ ë²ˆì§¸ ìµœì†Œ ê³ ìœ ë²¡í„°(Fiedler ë²¡í„°)ë¥¼ ì´ìš©í•œ ì™„í™”ë¡œ ê·¼ì‚¬í•  ìˆ˜ ìˆìŠµë‹ˆë‹¤.

Rayleigh ëª«:

$$\min_{\mathbf{y}} \frac{\mathbf{y}^T L \mathbf{y}}{\mathbf{y}^T D \mathbf{y}} \quad \text{s.t. } \mathbf{y}^T D \mathbf{1} = 0$$

í•´ëŠ” ì¼ë°˜í™” ê³ ìœ ê°’ ë¬¸ì œ $L \mathbf{y} = \lambda D \mathbf{y}$ì˜ ë‘ ë²ˆì§¸ ìµœì†Œ ê³ ìœ ë²¡í„°ì…ë‹ˆë‹¤.

def spectral_clustering(A, n_clusters=2):
    """
    ìŠ¤í™íŠ¸ëŸ¼ êµ°ì§‘í™” ì•Œê³ ë¦¬ì¦˜

    Parameters:
    -----------
    A : ndarray
        ì¸ì ‘ í–‰ë ¬
    n_clusters : int
        êµ°ì§‘ ìˆ˜

    Returns:
    --------
    labels : ndarray
        ê° ì •ì ì˜ êµ°ì§‘ ë ˆì´ë¸”
    """

    # ê³ ìœ ê°’ ë¶„í•´ (ìµœì†Œ n_clustersê°œì˜ ê³ ìœ ë²¡í„°)

    # ìµœì†Œ n_clustersê°œì˜ ê³ ìœ ë²¡í„° ì„ íƒ

    kmeans = KMeans(n_clusters=n_clusters, random_state=42)
    labels = kmeans.fit_predict(U_normalized)

    return labels, U

def pagerank(A, d=0.85, max_iter=100, tol=1e-6):
    """
    PageRank ì•Œê³ ë¦¬ì¦˜

        ì¸ì ‘ í–‰ë ¬

        Damping factor

    Parameters:

        ì¸ì ‘ í–‰ë ¬
    signal : ndarray
        ê·¸ë˜í”„ ì‹ í˜¸

        ì£¼íŒŒìˆ˜ ì˜ì—­ ì‹ í˜¸

def gcn_layer(A, H, W, activation=lambda x: np.maximum(0, x)):
    """
    GCN ë ˆì´ì–´ êµ¬í˜„

    Parameters:
    -----------
    A : ndarray, shape (n, n)
        ì¸ì ‘ í–‰ë ¬

        ì…ë ¥ íŠ¹ì§• í–‰ë ¬
    W : ndarray, shape (d_in, d_out)
        ê°€ì¤‘ì¹˜ í–‰ë ¬
    activation : function

    A_tilde = A + np.eye(n)

    ê°„ë‹¨í•œ ê·¸ë˜í”„ ì–´í…ì…˜ ë ˆì´ì–´
